# AI Resume Screening & ATS Analytics System

## 📌 Project Overview
This project is an AI-powered Resume Screening and ATS Analytics System built using Python, Streamlit, Machine Learning, NLP, and Power BI.

It helps analyze resumes, extract skills, calculate ATS score, predict job roles, and visualize recruitment insights using dashboards.

## 🚀 Features
- Resume PDF Upload
- Resume Text Extraction
- Skill Extraction
- ATS Score Calculation
- Matched Skills and Missing Skills
- ML-based Job Role Prediction
- Candidate Ranking
- Power BI Dashboard Gallery
- Professional Streamlit UI

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- PDFPlumber
- Machine Learning
- NLP
- Power BI
- GitHub
